#**Manali Airbnb Budget Analysis**

**Project Overview**

This project analyzes a dataset of 12,000+ Airbnb listings in Manali to identify the best budget-friendly stays (under 5,000 INR). By using custom metrics like a Refined Value Score, the analysis identifies properties that offer the highest quality relative to their price.

**Key Features**

**Data Cleaning:** Handled missing values and standardized "New" listing ratings using median values.

**Feature Engineering:** Extracted guest counts from text data and created binary flags for key amenities like Mountain Views and Wifi.

**Visual Insights:** Includes a Correlation Heatmap, Neighborhood Price Distribution (Box Plot), and a "Sweet Spot" Scatter 

**Installation & Usage**

**Prerequisites:** Python 3.x, Pandas, Matplotlib, Seaborn.

**Execution:** Open Airbnb_Analysis.ipynb in Google Colab or Jupyter Notebook.

**Data Source:** The analysis uses a scraped dataset of Manali Airbnb listings.
